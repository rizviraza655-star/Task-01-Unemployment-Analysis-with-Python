📊 Unemployment Analysis with Python

📌 Project Overview

This project analyzes the unemployment rate in India using Python. The objective is to clean the dataset, explore unemployment trends, understand the impact of COVID-19, identify seasonal patterns, and generate insights through data visualization.

This project was completed as **Task 2** for the **CodeAlpha Data Analytics Internship**.

---

🎯 Objectives

* Import and clean the unemployment dataset.
* Perform exploratory data analysis (EDA).
* Analyze unemployment trends across different states.
* Study the impact of COVID-19 on unemployment rates.
* Identify monthly and seasonal unemployment patterns.
* Visualize important insights using charts and heatmaps.
* Generate conclusions that can help in economic and social policy decisions.

---

🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab / Jupyter Notebook

---

📂 Dataset

The project uses the **Unemployment in India** dataset containing:

* Region (State)
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)

---

📈 Analysis Performed

* Dataset overview and inspection
* Missing value detection and data cleaning
* Data type conversion
* Descriptive statistics
* State-wise average unemployment analysis
* Monthly unemployment trend analysis
* COVID-19 unemployment impact analysis (2020)
* Seasonal trend analysis
* Top 10 states with highest unemployment rates
* Correlation analysis using a heatmap

---

📊 Visualizations

The notebook includes:

* Bar Chart of Average Unemployment by State
* Monthly Unemployment Trend Line Chart
* COVID-19 Impact Line Plot
* State vs Month Heatmap
* Seasonal Trend Line Graph
* Top 10 States Bar Chart
* Correlation Heatmap

---

🔍 Key Insights

* Unemployment rates increased significantly during the COVID-19 pandemic.
* Lockdown months recorded the highest unemployment levels.
* Certain states consistently experienced higher unemployment than others.
* Seasonal variations can be observed across different months.
* Employment indicators show meaningful relationships that can assist policymakers in understanding labor market conditions.
